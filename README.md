## ☕ Ordering System

A command-line-based food ordering system that simulates a simple billing workflow.

### 🔹 Features
- Menu display with item prices
- User input for selecting items
- Subtotal calculation
- Tax calculation (15%)
- Final order summary

### 🔹 Tech & Concepts
- Python
- Lists & Dictionaries
- Functions and modular design
- Input handling
- Business logic implementation

---

## 📂 File Operations

A set of utilities for handling and processing text files.

### 🔹 Features
- Read entire file content
- Convert file into list of lines
- Extract and write first line to a new file
- Read even-numbered lines
- Reverse file content

### 🔹 Tech & Concepts
- Python file handling (read/write)
- String manipulation
- List operations
- Enumerate & loops

---

## ▶️ How to Run

### Run Ordering System

- cd ordering-system
- python main.py


### Run File Operations

- cd file-operations
- python file_ops.py
